# AI Automation Portfolio

A confidentiality-conscious portfolio presenting concrete AI automation work, built by a hybrid **business / AI automation** profile.

This repository is intended to show *how* I approach automation problems — the business logic, technical reasoning, stack, and architecture behind my projects — without exposing sensitive workflows, private prompts, credentials, client data, or full automation exports.

## Why This Portfolio Is Confidentiality-Conscious

These projects involve business logic, client context, and operational know-how that should not be public. This portfolio is deliberately designed to communicate capability **without** disclosing anything sensitive.

You will **not** find here:

- Private prompts or full prompt libraries
- API keys, credentials, or secrets
- Full workflow exports (e.g. complete n8n JSON)
- Client data or identifiable business information
- Internal sourcing logic or proprietary scoring rules

Instead, each project documents the **architecture, business case, technical approach, and roadmap** at a level that demonstrates the work while keeping the sensitive parts protected. Each project also includes a dedicated `what-is-not-public.md` file that is explicit about what has been intentionally omitted and why.
